# ConsoleAppHospital

A simple console application for managing hospital employees.  
Built as part of the **Kampa Play** challenge: https://www.youtube.com/watch?v=XRlCUK7iOco&t=314s

## Features

- Add, remove and view employees
- Supports three job titles: **doctor**, **nurse**, **custodian**
- Saves employee data to local text files
- Automatically loads employees on startup

## Commands

| Command  | Description                        |
|----------|------------------------------------|
| `add`    | Add a new employee                 |
| `remove` | Remove an employee by ID           |
| `view`   | View an employee by ID             |
| `load`   | Load employees from files          |
| `page`   | Page all medical employees         |
| `help`   | Show available commands            |
| `exit`   | Exit the application               |

## Requirements

- .NET 10.0
